Pasos para la ejecución del repositorio

1. Descargamos el repositorio

2. Descomprimimos y abrimos anaconda.navigator (Importante actualizar a la última versión para que funcione el modelo ya que esta versión actualizada trae una dependencias necesarias por lo que se es necesario tener la última versión) en este caso, sin embargo, se puede realizar por consola en caso de que se use el entorno visual

3. nos vamos al apartado de Enviroment 

4. le damos en importar y abrimos la ubicación del repositorio y elegimos el archivo Enviroment.yml 

5. importamos el ambiente y esperamos que cargue 

6. una vez cargue ponemos abrir el jupyter en este caso le damos clic al ambiente y abrimos la consola de ipython y nos dirigimos a la carpeta de yolov5

7. Instalamos los requerimientos con la siguiente línea: pip install -r requirements.txt

8.  ves cargue ponemos abrir el jupyter en este caso le damos clic al ambiente y abrimos con jupyter

9. accedemos al la ubicacion donde esté guardado el repositorio y acedemos a la siguiente ruta  /yolov5/Soporte.ipynb

10. se puede ver el estado de la última ejecución o si desea ver los videos puede ejecutar de nuevo
